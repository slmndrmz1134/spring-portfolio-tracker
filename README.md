# Spring Portfolio Tracker 📈

A robust RESTful backend service built with Java and Spring Boot for tracking stock market portfolios. The application integrates seamlessly with the Yahoo Finance API to fetch real-time market data and uses MySQL for reliable data persistence.

## 🚀 Features
* **Real-Time Data:** Live stock price tracking via Yahoo Finance API.
* **RESTful Architecture:** Clean and scalable API endpoints for portfolio management.
* **Database Integration:** Relational data management using MySQL and Spring Data JPA.
* **Efficient Connection Pooling:** Utilizes HikariCP for optimal database performance.

## 💻 Tech Stack
* **Language:** Java 17
* **Framework:** Spring Boot 4.x
* **Database:** MySQL
* **ORM:** Hibernate / Spring Data JPA
* **External Service:** Yahoo Finance API
* **Build Tool:** Maven
